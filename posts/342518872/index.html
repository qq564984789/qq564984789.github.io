Zephyr的安装

什么是Zephyr?

项目的取名，往往包含着作者的意图构想。zephyr的中文释义是微风、和风，这初看起来没什么特别。当我查了英英释义之后，发现它的意思是a slight wind (usually refreshing)，因此可以猜测作者希望这个项目可以让你如沐春风。简单地说，Zephyr是Intel主导的一个面向IOT的开源RTOS。它是Linux基金会的合作项目，主要针对资源受限、有无线连接需求的小型设备，目前支持超过200多种板子。在我看来，由于是Linux在大力推动，因此备受关注。下面主要介绍Zephyr在Winodows平台上的安装。

动手实验

上手Zephyr开发环境的搭建，大概需要以下3个步骤：

  1. 在主机上采用命令行环境：安装包管理工具、编译器、构建系统等工具。
  2. 获得源码。
  3. 在目标板上编译、下载、运行一个例程。

根据上述描述，下面具体分析这三个步骤:

一、选择、更新OS

Zephyr开发环境依赖于一个最新的主机系统、常用的编译构建系统等工具。首先，确保你开发环境的OS已经更新了。在Wiondows中，你可以手动检查更新，通过这样的方法：开始->设置->更新&安全->Windows更新，然后选择更新检查。如果可以获取到新的更新，那就动手安装。

二、安装依赖

接下来，使用包管理器来安装开发环境所需要的其他工具。Python3和它自己的包管理器pip在Zephyr的安装和运行脚本里大量使用，帮助编译、构建和运行Zephyr应用程序。下面的指令都默认你使用的是cmd.exe工具。如果你使用的是PowerShell, 那么一些细节比如环境变量的设置也许会有不同。

安装Windows本地依赖的一个简单办法就是首先安装 Chocolatey ,一个Windows里的包管理器。如果你更喜欢手动安装依赖，你也可以从各自的网站上下载这些依赖，然后检查它们最后是否出现在你的环境变量的PATH里。

  1. 安装Chocolatey:可以按照 Chocolatey install 这个页面的指令来安装；

  2. 以管理员方式打开命令行工具（cmd.exe）

  3. 禁止全局确认，避免安装每个依赖的时候都要确认：

    choco feature enable -n allowGlobalConfirmation

​ 运行结果如下如所示：

  1. 安装CMake
    choco install cmake --installargs 'ADD_CMAKE_TO_PATH=System'

​ 运行结果如下如所示：

  1. 安装剩下的其他工具：
    choco install git python ninja dtc-msys2 gperf

  1. 关闭管理员方式运行的命令行，打开普通方式运行的命令行

  2. 安装west:

    pip3 install west

三、获取源码

Zephyr的多功能west工具，它简化了获取Zephyr项目的git仓库和Zephyr使用的外部模块这个过程。在一个新的zephyrproject路径下，使用下面的west指令，来拷贝Zephyr所有的git仓库。

    cd %HOMEPATH%
    west init zephyrproject
    cd zephyrproject
    west update

四、安装所需的Python包

我们下载的Zephyr源码下source目录下有一个requirments.txt文件，我们将使用这个文件来安装Zephyr工程使用的其他Python工具：

    pip3 install -r %HOMEPATH%\zephyrproject\zephyr\scripts\requirements.txt

五、安装软件开发工具链

构建Zephyr应用程序的工具链包含：编译器、汇编器‘、连接器以及它们的依赖。

  1. Windows平台并不支持Zephyr的SDK。安装第三方工具链的指令请点击这里.

  2. 不要忘记设置环境变量（ ZEPHYR_TOOLCHAIN_VARIANT 、工具链特定的名称等）来告诉构建系统到哪里去寻找工具链。

六、编译Blinky程序

例程 Blinky Application 的功能是让目标板上的一个LED闪烁。通过编译运行这个例程，我们可以验证Zephyr的开发环境和工具是否已经正确地搭建好。

  1. 设置编译的环境变量：
   cd %HOMEPATH%/zephyrproject/zephyr
   zephyr-env.cmd
  1. 编译Blinky程序，然后指定你板子的名称(具体板子名称请看 Supported Boards )：
   west build -p auto -b <your-board-name> samples/basic/blinky

west命令使用-p auto参数来自动清除上一次编译命令产生的副产品。

七、下载、运行程序

  1. 用一根USB线来连接板子和你开发用的电脑。
  2. 如果板子上有开关，就把它打开。
  3. 使用下面的指令来下载例程：
    west flash

应用程序会开始运行，然后你就能看到即刻的闪烁。闪烁的LED，它所在的实际位置是和板子有关的。
